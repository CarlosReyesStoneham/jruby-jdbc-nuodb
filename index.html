Welcome to the NuoDB JRuby JDBC Driver Project

This site contains details on how to install and use the NuoDB JRuby JDBC Driver. To install the driver simply run the following command for pre-release builds:

$ gem install jdbc-nuodb --pre

For release builds it's simply:

$ gem install jdbc-nuodb
